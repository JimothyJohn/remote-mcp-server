Available Tools

Ready-to-use tools that AI assistants can leverage for enhanced functionality

hello_world

hello_world(name: str = "World") → str

Generates personalized greeting messages with customizable names. Perfect for testing connectivity and demonstrating basic tool functionality.

get_current_time

get_current_time() → str

Returns the current timestamp in ISO format. Essential for time-sensitive operations and logging with precise temporal information.

echo_message

echo_message(message: str, repeat: int = 1) → str

Echoes messages with optional repetition (max 10x). Useful for testing, debugging, and message amplification scenarios.

get_server_info

get_server_info() → dict

Provides comprehensive server status, metadata, and health information. Critical for monitoring and system diagnostics.

calculate_sum

calculate_sum(numbers: list[float]) → float

Performs mathematical summation of numeric arrays (up to 100 numbers). Demonstrates computational capabilities and input validation.
